# Телефонный справочник (Task 1 — file storage)

## Описание

Проект: настольное приложение «Телефонный справочник» на C++17.

На этом этапе (**Task 1**) реализована консольная версия без Qt:

- контакты хранятся в текстовом файле `contacts.txt`;
- данные загружаются при старте и сохраняются при выходе;
- есть простое текстовое меню для работы со справочником.

Графический интерфейс на Qt и работа с PostgreSQL будут добавлены на следующих задачах.

## Основные классы

- `Contact` — один контакт (имя, фамилия, отчество, адрес, дата рождения, email, номера телефонов).
- `PhoneNumber` — номер телефона и его тип (рабочий, домашний, служебный).
- `ContactRepository` — интерфейс хранилища контактов.
- `FileContactRepository` — реализация хранилища в текстовом файле (`contacts.txt`).
- `validation` — функции проверки корректности полей через регулярные выражения.
- `ConsoleApplication` — консольное приложение с текстовым меню (загрузка, сохранение, операции над списком контактов).

## Функциональность Task 1

Консольное приложение позволяет:

- загружать список контактов из файла при старте;
- выводить все контакты на экран;
- добавлять новый контакт;
- изменять существующий контакт;
- удалять контакт;
- выполнять поиск по имени, фамилии или email;
- сортировать контакты по фамилии, имени или email;
- сохранять актуальное состояние списка обратно в `contacts.txt`.

Для каждого контакта проверяются:

- имя и фамилия — только допустимые символы, с учётом правил начала и окончания строки;
- email — корректный формат с латинскими символами и цифрами;
- телефон — поддерживаются заданные в условии варианты записи;
- дата рождения — корректная дата, не позже текущей.

## Сборка и запуск

В корне проекта:

```bash
make
./build/phonebook
```

При первом запуске файл `contacts.txt` будет создан автоматически (если его ещё нет).
